# 🐾 Amigo Fiel - Plataforma de Adoção de Pets

O **Amigo Fiel** é um ecossistema web desenvolvido para conectar animais em situação de vulnerabilidade a novos lares. O projeto foca em uma experiência de usuário (UX) intuitiva, acessibilidade e organização modular de código, servindo como uma solução prática de impacto social.

## 👥 Desenvolvimento Colaborativo

O projeto foi construído em equipe, utilizando metodologias de divisão de responsabilidades para garantir a eficiência e a qualidade do código:

* **Arquitetura e Estruturação (HTML5):** [Arthur Vinícius](https://github.com/thurr-vini)
* **Design e Estilização (CSS3):** [Augusto Souza](https://github.com/AugustoCGM)
* **Lógica e Interatividade (JavaScript):** [Joab Pessoa](https://github.com/JoabPessoa)

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Layout responsivo (Grid/Flexbox), Glassmorphism e Dark Mode.
* **JavaScript (ES6+):** Manipulação dinâmica do DOM e lógica de filtragem.
* **Web Storage API:** Persistência de preferências do usuário (localStorage).

## 🧠 Destaques da Estruturação (HTML)

Sob responsabilidade de **Arthur Vinícius**, a base do projeto foi desenvolvida focando nos seguintes pilares técnicos:

1.  **Semântica Avançada:** Uso rigoroso de tags estruturais (`<header>`, `<main>`, `<section>`, `<footer>`) para otimização de SEO e acessibilidade.
2.  **Validação de Dados:** Implementação de formulários inteligentes com tipos de entrada específicos e máscaras via `pattern`, garantindo a integridade da captação de leads de adoção.
3.  **Acessibilidade (A11y):** Hierarquia de títulos e textos alternativos em mídias para navegação inclusiva.

## ⚙️ Lógica e Interatividade (JS)

A camada de inteligência, desenvolvida por **Joab Pessoa**, inclui:

* **Renderização Dinâmica:** Criação automatizada de componentes (cards) a partir de uma base de dados interna.
* **Filtros Inteligentes:** Sistema de busca segmentada por espécie utilizando métodos de Array (filter/forEach).
* **Persistência de Tema:** Lógica de Dark Mode integrada ao armazenamento local do navegador.

## 🚀 Como Executar o Projeto

1.  Clone o repositório:
    ```bash
    git clone [https://github.com/AugustoCGM/Amigo-fiel.git](https://github.com/AugustoCGM/Amigo-fiel.git)
    ```
2.  Abra o arquivo `index.html` em seu navegador de preferência.
